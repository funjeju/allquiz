# 🏆 Rank & Quiz: The Viral Knowledge Battle Platform

## 📖 Project Overview
'Rank & Quiz'는 AI가 매일 생성하는 카테고리별(시사, 연예, K-pop, 지역 등) 데일리 퀴즈를 풀고, 
마이크로 랭킹(연령/지역별)을 쟁취하는 전 국민 지식 배틀 플랫폼입니다. 
핵심 성장 엔진은 **카카오톡 기반의 '도발적 바이럴 루프(Provocative Viral Loop)'**이며, 
Vercel OG(Open Graph) 기술을 활용해 실시간 랭킹이 반영된 동적 공유 이미지를 생성하여 신규 유저를 폭발적으로 인입시킵니다.

## ✨ Core Engines
1. **Dual-Core CMS:** AI 무인 출제(텍스트) + Human Studio(이미지/영상/틀린그림찾기 프로모션).
2. **Micro-Leaderboards:** 전국 단위가 아닌, "강남구 30대 K-pop 마스터" 수준의 세밀한 랭킹 부여.
3. **Hyper-Viral System:** Kakao Link API + Vercel Edge Functions를 활용한 실시간 랭킹 도발 초대장 발송.
4. **Gamified Ad-Tech:** 퀴즈 완료 후 보상(쿠폰)과 결합된 '브랜드 틀린그림찾기' 광고 모듈.

## 🛠 Tech Stack (Production Ready)
- **Frontend:** Next.js 14 (App Router), React, Tailwind CSS, shadcn/ui, Framer Motion (애니메이션)
- **Viral Tech:** Vercel OG (동적 공유 이미지 생성), Kakao JavaScript SDK (카카오링크 API)
- **Backend/DB:** Firebase (Firestore, Authentication, Cloud Functions), Vercel Edge/Serverless
- **AI Core:** Google Gemini 1.5 Flash (뉴스 텍스트 요약, 퀴즈 출제, 도발 멘트 생성)
- **Data Source:** Google News RSS Parser
