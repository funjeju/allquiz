# 🎨 UI/UX & shadcn/ui Components Spec

## 1. 결과창 & 도발 공유 뷰 (The Climax)
퀴즈 세션이 끝난 직후 유저의 도파민이 가장 높은 순간을 공략합니다.
- **Card (Result Dashboard):** 내 점수, 랭킹, 상위 퍼센테이지를 거대한 타이포그래피로 표시.
- **Alert (Temptation Nudge):** "내 순위가 불안한가요? 카톡 친구를 밟고(?) 순위를 올리세요!" 텍스트 노출.
- **Button (Kakao Share):** 커스텀 노란색 브랜드 컬러 적용. 클릭 시 카카오링크 API 모달 호출.
- **Drawer (Preview):** 카톡으로 상대방에게 어떻게 보여질지(OG 썸네일과 도발 멘트) 미리보기를 하단에서 올려서 보여줌.

## 2. 딥링크 인입 뷰 (The Hook)
카톡 링크를 누르고 들어온 신규/기존 유저를 위한 전용 랜딩 페이지.
- **Alert Dialog:** 화면 진입 즉시 팝업. "🔥 김**님의 도전장이 도착했습니다! 지금 즉시 퀴즈를 풀면 점수 1.2배 보너스!"
- **Progress:** 로딩 화면을 스킵하고 즉시 타임어택 퀴즈 화면으로 강제 이동(Frictionless Onboarding).

## 3. 틀린그림찾기 (광고 PPL 스테이지)
- **Aspect Ratio:** 광고주 이미지를 화면에 꽉 차게 렌더링.
- **Toast:** 제한 시간 15초 내에 3곳을 터치할 때마다 즉각적인 피드백 애니메이션(파티클 효과).
- **Dialog (Reward):** 성공 시 화려한 폭죽 애니메이션과 함께 실물 쿠폰 바코드 노출.

① 틀린 문제 강조 UI
- "🔥 이 문제 78% 틀림"
- "❌ 당신도 틀렸습니다"
② 타임어택 UI
- Progress Bar (5초 카운트다운)
- 색상 변화 (초록 → 빨강)
③ 문제 공유 버튼 위치 변경
- 문제 바로 아래
- 틀린 순간 즉시 노출
④ 비교 결과 카드
- 내 점수 vs 친구 점수
- 퍼센타일 표시
